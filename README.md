Este é um servidor NodeJs para você brincar com seus amigos.

=> Start Gamer:
 . node ./server.js

curiosidades:
 . Ligue o servidor em uma maquina, e fala assim pros seus amigos aquele que quebrar a senha do servidor, vai entrar pros anonymous.

=> Dependencas:
  . express
  . body-parser
  . dotenv
